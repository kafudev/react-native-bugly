# react-native-bugly

react-native bugly 异常上报和运营统计

## Installation

```sh
npm install @kafudev/react-native-bugly
```

## Usage

```js
import Bugly from "@kafudev/react-native-bugly";

// ...

const result = await Bugly.multiply(3, 7);
```
